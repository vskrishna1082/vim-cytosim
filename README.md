## vim-cytosim

Vim syntax highlighting and indentation for cytosim (`*.cym`) and cytosim 
preconfig (`*.cym.tpl`) files.

### Installation

- vim-plug

```
Plug 'vskrishna1082/vim-cytosim'
```

- vundle

```
Plugin 'vskrishna1082/vim-cytosim'
```
