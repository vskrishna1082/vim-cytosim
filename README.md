## vim-cytosim

Vim/Neovim syntax highlighting and indentation for cytosim (`*.cym`) and cytosim
preconfig (`*.cym.tpl`) files.

### Installation

- vim-plug

```
Plug 'vskrishna1082/vim-cytosim'
```

- vundle

```
Plugin 'vskrishna1082/vim-cytosim'
```

- lazy.nvim

```
{
	'vskrishna1082/vim-cytosim'
}
```
